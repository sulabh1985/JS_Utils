Javascript Utility Functions

Show/Hide by Element Id


To Show the Element
var util = Utility();
util.showElementById('showone');

To hide the Element
var util = Utility();
util.hideElementById('showone');

Show/Hide Elements by Multiple Ids


To Show the Elements
var util = Utility();
util.showElementByIds(['showtwo','showthree']);

To hide the Elements
var util = Utility();
util.hideElementByIds(['showtwo','showthree']);
